# txx

Unsafe interop between Deno FFI and C++

## Usage

See `examples/basic_usage.ts` for usage.

## Features

- Functions
- Classes
- Structs
- Enums
- Typedefs
- Basic class templates

## Not yet supported

- Partial class specializations
- Template aliases
- Namespaces with overlapping definitions: All namespaces are collapsed into
  one.
